## 💻 Sobre o Projeto

O Carrinho de Compras é uma aplicação web dinâmica que simula o fluxo de seleção de produtos em um e-commerce. O foco principal deste projeto é a manipulação de eventos e do DOM (Document Object Model) para criar uma experiência de usuário fluida e interativa.

Este projeto foi desenvolvido como parte do currículo do curso **Técnico em Desenvolvimento de Sistemas** da Escola **SENAI A. Jacob Lafer**.

---

## ⚙️ Funcionalidades e Conceitos Aplicados

Nesta etapa do desenvolvimento, foram implementados e consolidados os seguintes conceitos:

**Manipulação de DOM:** Alteração dinâmica de elementos HTML através do JavaScript (adicionar itens, atualizar valores e remover produtos).

**Lógica de Programação:** Uso de funções, arrays e objetos para gerenciar o estado do carrinho.

**Event Listeners:** Captura de cliques e interações do usuário para disparar ações em tempo real.

**Design Responsivo:** Interface construída com Bootstrap 5, garantindo que o carrinho funcione bem em computadores e dispositivos móveis.

**Estilização Avançada:** Uso de Variáveis CSS e técnicas modernas para um visual limpo e profissional.
